Universal Post Holder

SKU: SET-UNIVERSALPOS

7 configurations available · SKU from SET-UPH-20

The 360° swivel base adapter is designed to mount existing Ø1/2" post holders using an M6 cap screw, providing flexible rotational alignment and rapid system setup.

The adapter features an integrated magnetic base, which securely holds components in position on an optical breadboard during initial alignment. This significantly reduces setup time and allows easy repositioning before final fixing.

Constructed from high-quality anodized aluminum, the adapter ensures durability, corrosion resistance, and reliable performance in laboratory environments.

Features

  • Mounts to Ø1/2" post holders via M6 cap screw
  • 360° rotational adjustment for flexible alignment
  • Magnetic base for temporary positioning on breadboards
  • Enables quick and efficient optical system assembly
  • Made from anodized aluminum for durability
