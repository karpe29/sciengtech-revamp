# Homepage Color Style Options

Three interactive prototypes — same content structure, different color systems.

Open any file in a browser (double-click or drag into Chrome/Edge).

| File | Style | Best for |
|------|-------|----------|
| [homepage-option-a-dark-ruby.html](./homepage-option-a-dark-ruby.html) | **Dark Charcoal + Ruby** `#0E1118` / `#E11D48` | Canonical Gemini brand — authoritative, RFQ ruby CTAs |
| [homepage-option-b-clinical-light.html](./homepage-option-b-clinical-light.html) | **Clinical Light** `#F3F4F6` / white + ruby accents | Maximum spec/data readability, procurement committees |
| [homepage-option-c-midnight-cyan.html](./homepage-option-c-midnight-cyan.html) | **Midnight Navy + Laser Cyan** `#060B14` / `#06B6D4` | Defense/photonics aesthetic, laser-tech gravitas |

All three include: Hero · Institutional proof · 3-pillar matrix · Data proof · Upload CTA · Footer.

## Mega-menu navigation

Hover any main nav item (**Solutions**, **Components**, **Utilities**, **About**) to open a full-width dropdown:

| Nav item | Dropdown content |
|----------|------------------|
| **Solutions** | 3 deep-tech integration cards + footer link |
| **Components** | 4-column catalog (Lasers · Optics · Optomechanics · Vacuum/HV) + catalog footer |
| **Utilities** | 4 utility categories + RFQ side panel |
| **About** | Company · Engineering Hub · Pune HQ panel |

**Tip:** Use a desktop-width browser (>1024px). Mega-menus are hidden on mobile in these prototypes.

Open files locally: double-click the HTML file or refresh if already open (`Ctrl+F5`).

**Option A** includes a hero product carousel + horizontal hardware strip using images in `../assets/slides/`.
